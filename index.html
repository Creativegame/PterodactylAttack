Pterodactyl Attack prototype

Dec 30, 2012:

PteroAttack is being rebuilt in HTML Canvas, due to slowness of developing in OpenGL. I'm currently converting the old code to this.

This means the game is playable everywhere, the web browser (seen below) and on both Android and iOS.

You can't shoot yet in this version. But there are 30 baby pterodactyls that fly toward you and explode automatically at random.

I built an android stress test with 50 exploding baby pterodactyls, running at 50fps. The current demo shown below runs at 60fps on Android.

Iteration should be faster with this new implementation that's better suited for just putting 2d images on the screen.
